História da Web

(Voltar)

A internet teve origens na década de 1960, quando o Departamento de Defesa dos Estados Unidos desenvolveu a ARPANET, uma rede para comunicação militar.

Ela evoluiu ao longo das décadas, com contribuições significativas de cientistas e engenheiros ao redor do mundo. O modelo TCP/IP, estabelecido nos anos 1970, tornou-se o padrão para comunicação na rede.

World Wide Web (WWW) foi introduzida nos anos 1990, facilitando o acesso e compartilhamento de informações através de navegadores.

Desde então, a internet se expandiu globalmente, transformando-se em uma ferramenta vital para comunicação, comércio, entretenimento e educação.

Client x Server

(Voltar)

Servidor fornece serviços ou recursos, respondendo a solicitações. Cliente solicita serviços ou recursos do servidor, iniciando a comunicação. Ambos interagem para trocar informações em uma rede.

Cliente: É o lado que consome informações.

Servidor: Serve informações para os clientes.

HTML

(Voltar)

HTML significa Hypertext Markup Language e foi desenvolvido por Tim Berners-Lee.

É a linguagem padrão de marcação para a criação de páginas web.

Junto com o CSS e o JavaScript, HTML é essencial para a criação de páginas web interativas e visualmente atraentes.

Tags

(Voltar)

As tags HTML são elementos de marcação utilizados para estruturar o conteúdo de uma página web.

Aqui estão algumas das principais tags e seus significados:

  • i = Coloca em italico.
  • strong = Deixa em negrito.
  • p = Define um parágrafo de texto.
  • u = Sublinhar.
  • h1,h2...,h6 = Define cabeçalhos de diferentes níveis.
